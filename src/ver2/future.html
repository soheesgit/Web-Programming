단기 목표 (2025년)

백엔드 개발 기초 역량 강화

기간
2025년 1학기 ~ 2학기
우선순위
높음

핵심 목표

  • Java 언어 심화 학습 및 객체지향 프로그래밍 원칙 완벽 이해
  • Spring Boot 프레임워크 기본 개념 및 구조 학습
  • RESTful API 설계 및 구현 능력 향상
  • 데이터베이스 설계 및 SQL 쿼리 최적화 학습
  • Git 버전 관리 시스템 능숙하게 활용

실행 계획

  • 인프런 강의 "스프링 입문 - 코드로 배우는 스프링 부트, 웹 MVC, DB 접근 기술" 수강 및 실습
  • 토이 프로젝트 2개 이상 완성 (Spring Boot + JPA + MySQL 활용)
  • 알고리즘 문제 주 3회 이상 풀이 (백준, 프로그래머스)
  • 개인 블로그에 학습 내용 정리 및 공유 (주 1회 이상)

AI 기반 동네 추천 플랫폼 프로젝트 완성

기간
2025년 여름방학
우선순위
높음

프로젝트 개요

사용자의 성향과 선호도를 분석하여 맞춤형 동네를 추천하고, 해당 동네의 관광 코스와 맛집 등을 자동으로 생성해주는 서비스입니다. 현재 기획 및 데이터베이스 설계가 완료된 상태이며, 여름방학 동안 API 구현 및 배포까지 완료할 계획입니다.

구현 목표

  • 사용자 성향 분석 알고리즘 구현 (NCF 추천 알고리즘 활용)
  • 동네 데이터 수집 및 분석 (공공 API 활용)
  • 자동 일정 생성 기능 구현
  • 최적 경로 추천 기능 구현
  • AWS를 활용한 서비스 배포

기술 스택

  • Backend: Spring Boot, JPA, MySQL
  • Frontend: React.js (프론트엔드 개발자와 협업)
  • Infra: AWS EC2, RDS, S3
  • API: Kakao Map API, 공공데이터 포털 API

코딩 테스트 역량 강화

기간
2025년 연중
우선순위
중간

목표 수준

  • 백준 골드 레벨 달성
  • 프로그래머스 Level 3 문제 해결 가능
  • PCCP(코딩 전문역량 인증) 취득

학습 계획

  • 알고리즘 기초 이론 복습 (정렬, 탐색, 그래프, DP 등)
  • 주 3회 이상 문제 풀이 (난이도 점진적 상승)
  • 풀이한 문제는 블로그에 정리하여 복습
  • 스터디 그룹 참여 (알고리즘 스터디 모임 주 1회)

핵심 학습 자료

  • "이것이 취업을 위한 코딩 테스트다" 교재 완독 및 실습
  • 프로그래머스 코딩테스트 고득점 Kit 완료
  • 백준 단계별 문제 풀이

전문 분야 선택

백엔드 개발 내에서도 특정 분야에 전문성을 갖추는 것이 목표입니다. 현재 관심 있는 분야는 다음과 같습니다:

  • 대용량 트래픽 처리 및 성능 최적화
  • 분산 시스템 설계 및 구현
  • 클라우드 네이티브 애플리케이션 개발
  • 데이터 파이프라인 구축 및 빅데이터 처리

역량 강화 계획

  • 선택한 전문 분야 관련 컨퍼런스 및 세미나 정기 참석
  • 관련 오픈소스 프로젝트 적극 기여
  • 기술 블로그를 통한 지식 공유 및 커뮤니티 활동
  • 관련 분야 최신 기술 트렌드 지속적 학습

평생 학습 원칙

기술 분야는 빠르게 변화하므로, 지속적인 학습과 성장이 필수적입니다. 다음과 같은 원칙을 세워 꾸준히 성장하고자 합니다:

  • 매년 새로운 기술 또는 프레임워크 1개 이상 학습
  • 분기별 1권 이상의 기술 서적 독서
  • 연간 2회 이상의 기술 컨퍼런스 참석
  • 월 1회 이상 기술 블로그 포스팅
  • 주기적인 토이 프로젝트를 통한 실험적 기술 적용

관심 기술 분야

  • 서버리스 아키텍처 및 FaaS(Function as a Service)
  • AI/ML 기반 백엔드 시스템
  • 블록체인 기술 및 Web3
  • IoT 백엔드 시스템
