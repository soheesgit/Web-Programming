1️⃣ 프로젝트 수행 계획

1-1. 창업동아리 활동 – AI 기반 동네 추천 플랫폼

기간

2025.05.20 ~ 2026.02.28

형태

팀 프로젝트 (백엔드 3명 + 프론트엔드 1명)

역할

백엔드

핵심 목표:
  • 사용자 성향 분석을 기반으로 세부 행정동(동네) 단위 맞춤 여행지를 추천하는 웹 기반 플랫폼 개발
  • 네이버 API와 사용자 입력을 종합하여 자동 일정 및 이동 경로 생성 기능 구현
  • 감성 태그, 실내/외 분류, 가격대 필터 등 사용자 중심의 다양한 탐색 기능 탑재
  • 지역 소상공인 노출 확대 및 슬로우 트래블/웰니스 트렌드에 최적화된 체류형 여행 제공
활용 기술:
  • 백엔드: Spring Boot, JPA, MySQL
  • 데이터 수집 및 가공: 공공데이터포털(OpenAPI), 기상청 날씨 API, 네이버 API
  • AI 추천 엔진: NCF(Neural Collaborative Filtering) 기반 사용자–장소 임베딩 학습 및 다층 퍼셉트론 구조를 활용한 개인화 동네 추천
  • API 설계: RESTful API 구조, Swagger 문서화
  • 기타: GitHub 협업, Notion 기반 일정 관리, Figma UI 와이어프레임 공유

1-2. 개인 토이 프로젝트 2건 이상 완성 및 배포

기간

2025년 여름방학 ~ 2026년 겨울방학

형태

개인 주도형 백엔드 중심 풀사이클 프로젝트

목표:
  • 서비스 기획부터 DB 설계 → API 서버 개발 → AWS를 통한 실서비스 배포까지 엔드투엔드(End-to-End) 경험 획득
  • 간단한 프론트엔드 페이지(React 기반)도 직접 구성
  • GitHub에 전체 프로젝트 기록(이슈 관리, 커밋, 브랜치 전략 등)을 체계적으로 저장
  • Tistory 블로그에 회고 글 게시
프로젝트 주제:
  • 주제①: 위치 + 날씨 기반 실시간 동네 산책 루트 추천 웹앱
    • 공공데이터(OpenAPI), 기상청 API 연동 + 위치 좌표 기반 경로 추천 + AWS로 배포
  • 주제②: 오늘 뭐 먹지? – 랜덤 식당 추천 웹앱 개인 예산 관리 웹서비스 (가계부 API + 대시보드)
    • Spring Boot + MySQL 기반 REST API, 프론트는 React, AWS로 정적 파일 배포
활용 기술:
  • 백엔드: Spring Boot, JPA, Spring Security, MySQL
  • 프론트엔드: React.js, HTML/CSS
  • 기타: GitHub Issue 기반 일정 관리, Notion 문서화, Tistory 정리(회고 + 문제 해결 과정 기록)

2️⃣ 컴퓨터 사이언스 기초 및 심화 학습

『혼자 공부하는 컴퓨터 구조+운영체제』

학습 계획:
  • 주 3회 학습 → 각 단원별 개념 정리 + 문제 풀이
  • 7월 중에 학습 끝내기 목표
  • Notion 정리 + 티스토리 기록

2-2. 자료구조 & 알고리즘 학습

학습 플랫폼:

백준 / 프로그래머스

목표:

주 3회 이상 문제 풀이, 하루 2문제 이상

학습 범위:
  • 정렬, 탐색, 그리디, DFS/BFS, DP, 스택/큐, 해시, 트리
  • 시간복잡도 분석 및 개선 훈련
추가 계획:
  • 티스토리에 풀이 방식과 어려웠던 점, 로직 해설 정리
  • 2026 상반기 코딩 테스트를 목표로 실전 감각 유지

2-3. 데이터베이스 학습

학습 방식:
  • 실습 기반 RDB 설계 훈련 (정규화, ERD 작성, JOIN 등)
  • SQL 문법 연습 및 쿼리 최적화 실습
  • 트랜잭션, 인덱스, 잠금 등 고급 개념도 자격증 학습과 병행
활용 자료:

SQLD 자격증 교재 + 실제 DB 설계 예제 실습
