1️⃣ 프로젝트 수행 계획

1-1. 창업동아리 활동 – AI 기반 동네 추천 플랫폼

기간

2025.05.20 ~ 2026.02.28

형태

팀 프로젝트

역할

백엔드

사용 기술

Spring Boot, MySQL, React

활동 개요:

AI 기반 동네 추천 플랫폼 서비스를 기획하고 시스템을 구현하는 실전형 팀 프로젝트입니다.
소프트웨어학과 동급생 4명(백엔드 3, 프론트엔드 1)으로 구성된 팀이며, 모두가 초보자로서 천천히 저희만의 속도로 진행 중입니다.
5월 20일 창업동아리에 최종 선정되어, 산학협력단의 금전적 지원 및 아이템 피드백을 받으며 개발을 이어가고 있습니다.

서비스 개요:
  • 여행 목적과 선호 성향을 고려해 맞춤형 동네 추천 제공
  • 기상 데이터와 실내/야외 장소 필터링 로직 연동
  • 추천 동네 내 주요 장소 정보 한눈에 조회
  • 리뷰, 피드백 등 사용자 상호작용 기능 탑재
현재 진행 완료된 사항:
  • MySQL 기반 데이터베이스 + ERD 작성
  • GitHub 협업 환경 + 커밋 전략 설정
  • REST API 명세서 작성 + 요청/응답 JSON 예시 문서화
나의 백엔드 구현 담당:
  • DTO 및 Entity 구현 (진행중): 사용자 성향 및 장소 정보를 객체로 구조화하고, 로직 흐름에 맞게 설계 및 연동
  • 추천 로직 개발: Rule-based 필터링 및 Content-based Filtering을 결합한 추천 알고리즘 설계 및 구현 예정
  • 다중 조건 필터링: 장소의 카테고리, 실내/야외 여부, 시간대 적합성 등 다양한 조건을 고려하여 정교한 필터링 수행 예정
  • 향후 고도화 계획: 사용자 피드백 기반 추천 정확도 향상 및 추천 근거 설명 기능 구현 예정
최종 목표:
  • 실전 팀프로젝트 협업 경험 쌓기
  • 포트폴리오에 활용 가능한 프로젝트 완성

2️⃣ 컴퓨터 사이언스 기초 및 심화 학습

2-1. 운영체제 + 컴퓨터 구조 독학

학습 이유:

백엔드 개발자로서 시스템의 동작 원리를 이해하기 위해 운영체제 과목 학습이 필요하다고 판단함.
하지만 해당 과목을 이수하지 않아 운영체제에 대한 기반 지식 부족함.
따라서 이번 여름방학 중 7월 한 달 간 독학을 통해 핵심 개념 보완 목표

교재:

『혼자 공부하는 컴퓨터 구조+운영체제』

학습 계획:
  • 주 3회 학습 → 각 단원별 개념 정리 + 문제 풀이
  • 7월 중에 학습 끝내기 목표
  • Notion 정리 + Tistory 기록

2-2. 자료구조 & 알고리즘 학습

학습 플랫폼:

백준 / 프로그래머스

사용 언어:

파이썬

목표:

주 3회 이상 문제 풀이, 하루 2문제 이상

학습 범위:
  • 정렬, 탐색, 그리디, DFS/BFS, DP, 스택/큐, 해시, 트리
  • 시간복잡도 분석 및 개선 훈련
추가 계획:
  • Tistory에 풀이 방식과 어려웠던 점, 로직 해설 정리
  • 2026 상반기 코딩 테스트를 목표로 실전 감각 유지

2-3. 데이터베이스 학습

학습 이유:

백엔드 개발자로서 안정적이고 효율적인 데이터 처리를 위해서는 SQL 문법 이상의 데이터베이스 구조와 성능 원리에 대한 이해가 필수라 판단함.
이번 학기에 수강한 '데이터베이스론'에서 배운 내용을 복습하고자 SQLD 자격 취득을 목표로 설정함.

학습 목표:

단순히 SQLD 시험에 나오는 키워드를 외우는 데 그치지 않고,
어떤 쿼리가 어떤 상황에서 더 빠르게 작동하는지, 어떤 조건에서 성능이 악화되는지를
스스로 설명할 수 있는 수준의 컴퓨터공학적 기반 지식을 갖추는 것을 목표로 함.

교재:

『SQLD 자격검정 실전문제』

추가 계획:
  • SQLD 자격 취득 이후, 실제 프로젝트의 DB 설계에 적용해보기
  • MySQL 실습 환경을 직접 구축해 다양한 SQL 쿼리 테스트 진행
